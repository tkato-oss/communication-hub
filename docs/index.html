Security

安心のセキュリティ設計

お客様のデータを守るため、厳格なセキュリティ基準で設計・運用しています。

OAuth認証によるAPI連携

各チャットツールとはOAuth 2.0で安全に接続。パスワードをお預かりすることは一切ありません。接続権限はいつでも取り消し可能です。

データ非保持ポリシー

メッセージ内容をサーバーに保存しません。表示・送信時のみ一時的に処理し、処理完了後は即時破棄。情報漏洩リスクを最小化しています。

通信の完全暗号化

すべての通信はTLS 1.3で暗号化。ツール間のデータ中継を含め、平文での通信は一切発生しません。第三者による傍受を防止します。

アクセス制御

管理者による利用者の権限設定が可能。誰がどのツールに接続できるかを細かくコントロールでき、組織のセキュリティポリシーに準拠した運用を実現します。
